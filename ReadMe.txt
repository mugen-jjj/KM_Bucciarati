----------------------------------------------------------------------------------------------
キャラクター概要
----------------------------------------------------------------------------------------------
キャラ名：  KMブチャラティ
動作対応：  MUGEN1.1bとIKEMEN GOで動作確認済み
改変：      スプライトはyapoos氏ブチャラティのものを使用しています。

性能のコンセプトは「スピート重視のライバルキャラ」機動力のある技が多く器用に戦えます。
一気に距離を詰める【輝彩疾走】【KMダッシュ】からの派生技や加速する飛び道具【これはおれの腕ではない】を中心にペースを握り続けましょう。

22技の強化技【ブッ殺すと思った】は各種スキル技を強化します。
強化中のスキル技をヒット・ガード時させると、自動で専用の派生技がでるようになり、高威力のコンボを狙えます。
ギルティギアの紗夢の「朝凪の呼吸」ようなものと言えば通じるかもしれません。
各スキル技が強化状態かどうかは、上の3つの赤いアイコンの点灯が示しています。


ほか参考にしたキャラ：GBVSランスロット、スマブラSPサムス、GGチップ　など

基本的に技はヒットorガードさせればキャンセル可能です（飛び道具含む）。
方向キーの特殊技はキャンセルできるものとできないものが存在します。
